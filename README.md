## Prerequisite

```sh
- node
- watchman
- React Native: 0.68.2
- JDK: 11 (Azul Zulu)
- Xcode, iOS Simulator, & Command Line Tools
```

For further information: https://reactnative.dev/docs/environment-setup

## Install

```sh
yarn install
```

## Usage

To run on Android
```sh
yarn android
```

To run on iOS
```sh
yarn ios
```
